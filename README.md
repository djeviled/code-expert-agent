# Code Expert Agent

AI-powered code rescue service for broken AI-generated applications. We diagnose, repair, and deploy your code or you don't pay.

## Features

- **AI-Powered Diagnostics**: Deep static analysis to identify root causes of failure
- **Architecture-Level Repairs**: Strict typing, clean state management, dependency resolution
- **Production Deployment**: CI/CD wiring and environment configuration
- **"We Fix It Or You Don't Pay" Guarantee**: Success measured by functional, deployable applications
- **Terminal-Style Design**: Bricolage Grotesque and IBM Plex Mono fonts with emergency deployment theme
- **Status Indicators**: Critical failure status indicators and real-time diagnostics
- **3-Step Protocol**: Scan, Rebuild, Deploy process for systematic code rescue
- **Flexible Pricing**: Bundle vs upfront+balance payment models
- **Enhanced Chat Interface**: Diagnostic capabilities with real-time feedback
- **Comprehensive Pages**: About, Contact, Privacy, Terms, and Dashboard pages

## Tech Stack

- React/TypeScript frontend with Tailwind CSS
- Hono.js API framework
- Supabase for authentication and database
- Stripe for payment processing
- Anthropic Claude for AI agent
- Zo Space for hosting
